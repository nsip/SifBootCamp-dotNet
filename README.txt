################################################################################
# Summary
################################################################################

This project contains material used for the SIF Boot Camp conducted in November
2011 as part of the !dea11 Conference, Melbourne.

NOTE: The material associated with this project was updated 20/10/2013 to
remove references to the now deprecated SIFWorks ADK. Even though the
original Boot Camp used the SIFWorks ADK, it has now been superceded with the
OpenADK. The OpenADK is also now used in the SifCommonFramework and
SbpAgentFramework mentioned in the exercises documentation.

################################################################################
# Download Instructions
################################################################################

How to download this project:

Option 1 - As a ZIP file
========================
Click on the button marked "Download ZIP" available from the Code tab.

Option 2 - Using a Git client
=============================
From the command-line type: git clone git@github.com:nsip/SifBootCamp-dotNet.git

If you want to use this option but don't have the client installed, one can be
downloaded from http://git-scm.com/download.

Alternately, click on the button marked "Clone in Desktop" and follow the
instructions provided for installing a GitHub specific Git client.
